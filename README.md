# 더 버그 - 언더테이커: 곤충 로그라이크 프로토타입

2D 로그라이크 형식의 곤충 액션 게임 프로토타입입니다.  
스테이지에서는 외피를 교체·강화하며 일시적으로 성장하고, 마을에서는 진화 시스템을 통해 영구 능력치를 상승시킬 수 있습니다.

## 주요 시스템

- **재화 & 지갑(CurrencyWallet)**
  - 골드, 외피 조각, 날개 조각 등 여러 재화를 통합 관리
  - 드롭 아이템을 먹으면 자동으로 지갑에 반영

- **장비 & 외피 시스템**
  - 필드에 떨어진 외피 드롭을 줍기 → 즉시 장비 교체
  - 장착 시 현재 외피 타입 변경 및 플레이어 스프라이트 변경

- **진화 NPC (영구 성장)**
  - 공격력, 방어력, 추가 목숨, 추가 대쉬 등 영구 능력치 강화
  - ScriptableObject 기반 데이터 구조로 옵션·비용·최대 레벨 설정

- **외피 강화 NPC (탈피 성장)**
  - 스테이지 내에서 외피별 강화 레벨 상승
  - 재화 조건 체크 후 강화 진행

 - **무한 랜덤 스테이지**
   - 스테이지에 진입할때마다 맵 랜덤 생성
   - 보상으로 랜덤 외피 제공.
